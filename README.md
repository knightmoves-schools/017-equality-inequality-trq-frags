the Store.cs file
- should contain a method named `PriceCheck` that takes a `price` that is of type double and returns the string "match" if the price is equivalent to `1.50`
- should return an empty string ("") when 1.00 is passed in as an argument to `PriceCheck`

- should contain a method named `GuessNumber` that takes a `guess` that is of type int and returns the string "not it" if the number is not equivalent to `25`
- should eturn an empty string ("") when the number `25` is passed in as an argument to `GuessNumber`
